# 🎨 AI 智能抠图工具

基于 **U2-Net** 深度学习模型的一键抠图网站，自动去除图片背景，保留人物主体。

## ✨ 功能特性

- 🧠 **AI 智能识别** — 基于深度学习，精准识别人物主体边缘
- ⚡ **秒级处理** — 本地运行，几秒内完成抠图
- 🖼️ **高清导出** — 输出透明 PNG 图片
- 📤 **拖拽上传** — 支持点击或拖拽上传图片

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 浏览器打开 http://localhost:3000
```

## 使用方式

1. 打开网页后拖拽或点击上传图片（支持 JPG/PNG/WebP/BMP）
2. 点击「开始抠图」按钮
3. 等待 AI 处理完成，预览并下载结果

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Express.js + rembg (U2-Net) |
| 前端 | 原生 HTML/CSS/JS |
| 文件上传 | Multer |
